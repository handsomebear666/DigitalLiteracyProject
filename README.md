# 🛡️ 真相放大镜：相亲相爱一家人——防骗保卫战

## 📖 项目简介
这是一个基于原生前端技术（HTML/CSS/JS）开发的沉浸式交互数字素养科普小游戏。
玩家将扮演家族群里“全村的希望”，在名为“相亲相爱一家人”的模拟微信群中，通过互动对话、图片取证、链接查验等方式，极限救援即将落入诈骗陷阱的各位长辈。

本项目旨在用趣味、低门槛的交互方式，帮助中老年人提高数字素养，识别日常生活中的高发网络诈骗。

## ✨ 核心玩法与关卡
游戏高度还原了真实的微信聊天场景，并设计了三大典型高发诈骗关卡：
* **关卡一：二大爷的“变异巨兽” (AI 造假识破)**
  * **玩法**：利用放大镜功能，在图片中寻找不合常理的漏洞（如气候异常、扭曲的广告牌、AI 水印），引导长辈识别 AI 生成的虚假夺人眼球的文章。
* **关卡二：三姑的“免费羽绒服” (钓鱼网站拦截)**
  * **玩法**：进入高仿的促销网页，识破伪造的 URL 域名、制造焦虑的虚假倒计时，以及索要银行卡密码的致命陷阱。
* **关卡三：老妈的“百万保障客服” (深伪视频与屏幕共享危机)**
  * **玩法**：还原“百万保障”诈骗套路，识破非官方会议软件、虚假的“安全账户”，并紧急制止长辈交出“屏幕共享”权限。

## 💡 特色系统
* **沉浸式 UI**：高度还原微信聊天界面，包含真实的时间线计算、语音/表情图标等细节。
* **时光倒流引擎**：如果玩家在沟通中言辞过激（如嘲讽长辈）导致长辈逆反或被骗，系统会触发“闯关失败”弹窗，玩家可一键撤回错误操作，重新选择高情商的沟通方式。
* **无缝加载体验**：定制了微信风格的绿环 Loading 过渡屏，实现大体积多媒体资源的静默预加载。
* **视听反馈拉满**：包含微信原声提示音、点击音效，以及通关时的全屏动态礼花与音效。

## 🛠️ 技术栈
* **前端**：HTML5, CSS3, JavaScript (ES6)
* **架构**：纯原生数据驱动视图思路（无依赖框架），利用 DOM 操作与 CSS 动画构建完整交互闭环。
* **音频与多媒体**：H5 `Audio` API 懒加载策略，CSS3 `transform` 与 `keyframes` 实现粒子礼花特效。
